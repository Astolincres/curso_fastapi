# FastAPI Template

Este template implementa un CRUD en fastAPI

## Pre-requisistos

### Crear env

python -m venv venv

### Activar env

venv/Scripts/Activate.ps1

### Instalar dependencias

pip install -r .\requirements.txt

## Correr el proyecto

uvicorn main:app --reload
